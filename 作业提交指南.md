## 快速开始

### 1. 克隆仓库
```bash
git clone https://github.com/[your-username]/MiniTodoApp.git
cd MiniTodoApp
```

### 2. 使用Android Studio打开
- 打开Android Studio
- File → Open → 选择MiniTodoApp目录
- Gradle自动同步依赖

### 3. 编译和运行
```bash
# 编译debug版本
./gradlew assembleDebug

# 运行单元测试
./gradlew test

# 在模拟器上运行
./gradlew installDebug
```

### 4. 查看主要代码
- **主界面**: `app/src/main/java/com/example/minitodo/MainActivity.kt`
- **业务逻辑**: `app/src/main/java/com/example/minitodo/viewmodel/TodoViewModel.kt`
- **数据模型**: `app/src/main/java/com/example/minitodo/data/TodoEntity.kt`
- **提醒功能**: `app/src/main/java/com/example/minitodo/notification/AlarmUtils.kt`

### 常见问题解决

**问题1**: Gradle同步失败
**解决**: 
```bash
./gradlew clean
./gradlew sync
```

**问题2**: 编译错误 - Room数据库版本不匹配
**解决**: 确保使用了最新数据库版本（v3）

**问题3**: 运行时崩溃 - 权限不足
**解决**: 在Android 13+上手动授予通知权限
